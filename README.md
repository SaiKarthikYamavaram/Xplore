# How to use it ðŸ”§

From your command line clone, install and run app

```
# cloning this repository
$ git clone https://github.com/SaiKarthikYamavaram/Xplore

# Go into Repository
$ cd Xplore

# Install Dependencies
$ yarn install

# For android
$ react-native run-android

# For ios
$ react-native run-ios
```
